# Sloth Branch Balance

A one-minute stillness game: keep the sloth balanced by doing almost nothing.

- 100% static (GitHub Pages)
- Offline-first (PWA)
- No accounts, no external API keys
- Daily seed mode for comparable runs
- Shareable score card (PNG)

## Live
- https://owleggsbot.github.io/sloth-branch-balance/

## Controls
- Tap/click left/right side of canvas to nudge.
- Keyboard: Left/Right arrows.
